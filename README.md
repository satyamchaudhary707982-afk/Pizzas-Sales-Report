# 🍕 Pizza Sales Analysis Dashboard (Power BI)

Project Overview

This project focuses on analyzing pizza sales data using Power BI to uncover key business insights and trends. The dashboard provides an interactive and visual representation of sales performance, helping stakeholders make data-driven decisions.

Objectives

The main goal of this project is to analyze and visualize key performance indicators (KPIs) related to pizza sales, including:
Total Revenue – Overall earnings from pizza sales
Average Order Value (AOV) – Revenue per order
Total Pizzas Sold – Total quantity sold
Total Orders – Number of orders placed
Average Pizzas per Order – Customer purchase behavior

Dashboard Features

Sales Trends Analysis

Daily Trend of Orders – Identify fluctuations in daily sales
Hourly Trend of Orders – Detect peak business hours
Monthly Trends – Understand long-term performance patterns

Category & Size Insights

Sales by Pizza Category – Distribution of sales across categories
Sales by Pizza Size – Customer preference by size

Performance Analysis

Top 5 Best-Selling Pizzas (by Revenue, Quantity, Orders)
Bottom 5 Worst-Selling Pizzas – Identify underperforming items

Tools & Technologies

Power BI – Data visualization & dashboard creation
SQL / Excel – Data cleaning and transformation
DAX (Data Analysis Expressions) – KPI calculations

Key Insights

Identified peak hours and high-demand days for better resource planning
Highlighted top-performing pizza categories and sizes
Helped detect low-performing products for optimization
Improved understanding of customer ordering patterns
